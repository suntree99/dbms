# Where Clause

## Menggunakan Where
```sql
SELECT * FROM products WHERE quantity = 100;
```
```sql
SELECT * FROM products WHERE price = 15000;
```
```sql
SELECT * FROM products WHERE id = 'P0001'; 
```
```sql
SELECT * FROM products WHERE name = 'mie ayam bakso';
```
